## Compatibility

- Linux
- Windows
- Mac

## Installing

Install globally:

    npm install -g cocorico

## Running

With npm:

    npm install -g cocorico
    cocorico

Npx:

    npx cocorico

Node script:

    import cocorico from 'cocorico';

    cocorico();
